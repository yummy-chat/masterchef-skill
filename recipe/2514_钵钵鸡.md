# 钵钵鸡
所属菜系：川菜
烹饪难度：⭐⭐⭐
时令推荐：四季餐食

## 厨具与设备
🍳 平底锅（炒制干辣椒）
🔥 燃气灶
🔪 绞肉机（将炒好的干辣椒打成辣椒面）
🍲 汤锅（熬制高汤）
🥅 漏网（炸制白芝麻时使用）
🥄 汤勺（舀取热油、盛取高汤）
🥢 筅子（搅拌用）
🥣 不锈钢盆/玻璃碗（盛放辣椒面、辣椒油、汤底）
🪣 金属滤网（捞取香料残渣）
🌡️ 电子温度计（测量油温）

## 制作步骤
### 准备阶段
1. 将灯笼椒、子弹头、石柱红干辣椒剪成段；锅中倒入少量食用油，凉锅凉油下入辣椒段，开小火翻炒2分钟，转最小火再翻炒2-3分钟，至辣椒声音清脆、籽粒变为金黄色后盛出，彻底晾凉，用绞肉机打成中粗颗粒的辣椒面。
**关键点**：炒到辣椒声音清脆、籽变金黄即可。炒好后盛出彻底晾凉，用绞肉机打成中粗颗粒的辣椒面。
2. 将大葱切段、洋葱切丝、姜切片、大蒜拍扁；将所有干香料按配比备齐。
**关键点**：香料尽量凑齐，味道更好。
3. 称取100g生白芝麻备用。
4. 将鸡架冷水下锅焯水，煮至水面冒鱼眼泡时关火，捞出洗净；另取一汤锅加入3000ml清水，放入洗净的鸡架，加入葱段20g、姜片15g和高汤香料包，大火煮沸后转最小火煮75分钟。
**关键点**：鸡架焯水时间不宜过长；熬高汤时香料不能省略，是香味的来源。

### 制作阶段
1. 锅中倒入700g食用油，大火加热约2分钟至油温约110℃，下入所有准备好的葱姜蒜和干香料，保持大火炸5分钟，将材料炸至干枯金黄；转最小火，用漏网将所有料渣捞出丢弃。
**关键点**：捞料渣时务必转最小火，防止油温过高。
2. 油温升至约200℃时，保持最小火；将白芝麻倒入漏网，浸入油锅中，用筷子搅动，炸至芝麻呈淡金黄色且全部浮起，捞出滤干油分。
**关键点**：用漏网炸方便且易控。
3. 油温降至约190℃时，保持最小火；先舀一勺热油浇在辣椒面上搅拌均匀，再分次将剩余热油一勺勺浇入辣椒面中，边浇边搅拌，直至油全部浇完。之后盖上盖子焖24小时，风味更佳。
**关键点**：油温不宜过高，防止辣椒面炸糊；浇完油后盖上盖子焖24小时风味更佳。
4. 取一个大碗，依次加入用少许鸡汤化开的芝麻酱、芝麻香油15g、藤椒油35g、蒜末15g、红花椒粉1g、青花椒粉1g、鸡精15g、味精10g、盐20-25g、白糖25g、白胡椒粉0.5g，搅拌均匀；加入1000g煮好的高汤，再加入150g炸好的辣椒油和炸好的白芝麻，充分搅匀。
**关键点**：盐量要足，汤底需偏咸才能让串串入味；但汤底太咸会导致素菜发齁。
5. 将各种食材穿成串；利用锅中剩余的高汤烧开，先烫煮易熟的蔬菜约1分钟，再根据食材成熟时间煮制其他荤菜。所有煮好的串串放入调好的汤底中浸泡20分钟后即可食用。
**关键点**：鸡爪已随高汤煮熟，穿好串后可直接放入汤底中浸泡。

## 健康建议
这道钵钵鸡高油高钠，适合偶尔食用，不建议作为日常正餐。鸡架高汤营养丰富，但蔬菜串不易浸泡过久，以免吸油吸盐过多。
