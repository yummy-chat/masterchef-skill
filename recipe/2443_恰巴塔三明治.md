# 恰巴塔三明治
所属菜系：其他  
烹饪难度：⭐⭐⭐⭐  
时令推荐：秋季（9-10月）  

## 食材与调味品

🧂 盐 10g

## 制作步骤
### 基础面包制作
1. 制作波兰种：将100g面粉与100g水混合均匀，于室温下发酵至表面充满气泡呈糊状。
2. 制作主面团：将发酵好的波兰种与300g高筋粉、260g水、10g盐、1g酵母混合，揉成光滑面团，发酵至体积明显增大。
3. 整形与烘烤：将发酵好的面团分割整形，放入烤箱烘烤至表面金黄、体积膨胀。
4. 冷冻保存：面包完全冷却后，用保鲜袋密封，置于冷冻室保存。
   **关键点**：不可冷藏，冷藏会加速面包老化。
5. 解冻复烤：食用前从冷冻室取出面包，先在水龙头下快速冲水，再放入预热好的空气炸锅或烤箱复烤数分钟，至外皮酥脆。
   **关键点**：冲水与复烤可恢复接近刚出炉的口感。

### 烟熏三文鱼三明治
1. 将复烤后的恰巴塔从中间切开，切面均匀涂抹一层奶油奶酪。
2. 依次铺上极细红葱头丝、芝麻菜、牛油果薄片，最后铺上烟熏三文鱼片。
3. 撒上现磨黑胡椒即可。

### 酸奶水果三明治
1. 将复烤后的恰巴塔从中间切开，切面均匀涂抹希腊酸奶。
2. 将火晶柿子的果肉挖出，如同果酱般涂抹在酸奶层上。
3. 依次铺上切块的无花果和蓝莓。
4. 用削皮刀将黑巧克力削成碎屑撒在水果上，刨上帕玛森芝士碎，点缀杏仁片和薄荷叶。

### 泡菜牛肉三明治
1. 将切开的恰巴塔放入平底锅，两面喷少许橄榄油，小火加热约1-2分钟至表皮酥脆，盛出备用。
2. 锅中再放少许油，放入雪花肥牛片，煎至两面变色后立刻盛出。
   **注意**：牛肉煎制时间不宜过长，保持嫩滑口感。
3. 锅中重新放油，炒香洋葱丝，加入韩式辣酱和泡菜，翻炒出香味。
4. 将煎好的牛肉片倒回锅中铺在泡菜洋葱上，铺上芝士片，转最小火，盖上锅盖焖至芝士完全融化。
5. 将锅中融化的芝士牛肉馅料整块滑出，放在加热好的面包上，盖上另一半面包即可。

## 健康建议  
恰巴塔三明治搭配烟熏三文鱼、酸奶水果或泡菜牛肉，提供优质蛋白质和健康脂肪，适合健身或减脂人群选择。注意冷冻面包复烤前冲水能恢复口感，但需控制奶油奶酪或芝士片的用量以避免热量超标。
