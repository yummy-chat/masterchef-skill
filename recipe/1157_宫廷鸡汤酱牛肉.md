# 宫廷鸡汤酱牛肉
所属菜系：北京菜
烹饪难度：⭐⭐⭐
时令推荐：四季餐食

## 制作步骤

### 准备阶段
1. **备制"爆腌"香料**：将小茴香、大料、花椒放入无油锅中，小火干炒至出香味。炒好后盛出，用擀面杖稍加碾碎，然后与盐混合均匀。
- 全程小火，防止炒糊；炒香后再碾碎，风味更佳。

2. **处理牛肉**：将牛腱子肉洗净，用尖锐物在肉上扎一些孔，并用刀划开几处深切口。
- 扎孔和划口是为了破坏筋膜，让后续腌制和炖煮时更易入味。

3. **"爆腌"入味**：将混合好的盐和香料均匀涂抹、揉搓在牛肉的每个部位，放入盆中腌制1小时。
- 盐量要足，腌制时间至少1小时，此为快速入味的关键。

4. **清洗腌肉**：腌制好后，用清水将牛肉表面的腌料和血水冲洗干净。
- 必须洗净，否则后续炖煮汤汁易过咸且浑浊。

### 制作阶段
1. **炒制"油水糖色"**：锅中放少量油和水，加入冰糖，小火慢熬。期间用勺子不断搅动，观察糖液变化。待糖液从大泡转为密集小泡，颜色变为酱油色并冒青烟时，迅速放入大量葱姜翻炒，随即冲入温水制成糖色水。
- 全程小火，从下糖到完成约2-3分钟。
   - 糖色变为深琥珀色、冒青烟是临界点。
   - 油水混合法更易控制；下料和冲水时机要准，防止糖色变苦或溅油。

2. **焯水去腥**：将清洗干净的牛肉放入冷水锅中，大火加热。水开后转中小火，撇去浮沫。期间可加入少量凉水，反复几次，保持锅内"似开非开"的状态，直到不再有大量浮沫产生，将牛肉捞出洗净。
- 中小火焯水，时间视牛肉出血水情况而定。
   - 水面浮沫由多变少、变白，汤色转清即可。
   - 冷水下锅；焯水时加凉水，利用温差逼出更多血污；火不宜过猛，避免肉质变柴。

3. **高压锅炖煮**：将焯好水的牛肉放入高压锅，倒入足量鸡汤没过牛肉。放入用纱布袋装好的六种香料包、葱姜块、生抽、酱油、黄酒、胡椒粉，最后加入炒好的"葱姜糖色"水。
- 鸡汤是提鲜增香的灵魂；糖色水主要用于上色。

4. **压制与焖制**：盖上高压锅盖，上汽后转中小火压制40分钟。关火后不要开盖，利用余温再焖1小时。
- 上汽后压40分钟，关火焖1小时。
   - "压"+"焖"的组合能确保牛肉充分软烂且入味，又不至于过于散碎。

5. **出锅**：开盖后，用筷子能轻松扎透牛肉即可。将牛肉和汤汁一起倒出，晾凉后即可食用或冷藏保存。
- 牛肉在汤汁中浸泡晾凉会更入味。

## 健康建议
这道菜高蛋白低脂肪，牛腱肉富含铁质，适合健身或需要补血的人群。但钠含量较高（盐、酱油、生抽），高血压患者需控制食用量。建议搭配蔬菜如凉拌黄瓜或清炒时蔬以平衡营养。
