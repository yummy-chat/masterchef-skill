# 水煮肉片
所属菜系：川菜
烹饪难度：⭐⭐⭐⭐
时令推荐：四季餐食

## 制作步骤

### 准备阶段
1. **制作刀口辣椒**
   干辣椒剪去根部，保留辣椒籽。锅中放入少许食用油，小火下干辣椒与花椒粒，慢慢煸炒至辣椒颜色变深、花椒呈紫红色，立即盛出晾凉。待凉透后用刀切碎或打成粗颗粒备用。
**关键点**：全程小火，防止炒糊发苦。

2. **处理配菜**
   所有配菜洗净，按需要切好，总重量控制在一斤以内。

3. **切小料**
   生姜切末，葱白切葱花。大蒜取3瓣切成片，剩余蒜瓣拍扁后切细末，蒜片与蒜末分开存放。

4. **腌制肉片**
   通脊肉去除筋膜，用厨房纸吸干表面水分，逆着纹理切成约一个半硬币厚的片。加盐抓匀，打入整颗鸡蛋，捏碎蛋黄后朝同一方向不停搅拌约3分钟，直至肉片发干发黏、盆底出现胶质。将调好的浓稠湿淀粉分次加入，继续朝同一方向抓匀上劲，静置备用。
**关键点**：逆纹切片保持嫩度；充分搅拌至起胶上劲；用湿淀粉上浆更滑嫩。

### 制作阶段
1. **炒配菜垫底**
   锅中留少许底油，下所有配菜大火快速翻炒，加约1克盐调底味。炒至蔬菜变软蔫塌，盛出铺在大碗底部。

2. **炒制汤底**
   锅中另放正常炒菜量的油，油温微热时转最小火，下剁细的豆瓣酱，小火慢炒至油色红亮、香气溢出。加入葱花、姜末、蒜片炒香，再加入两大勺刀口辣椒炒香。倒入两碗半清水，转大火煮开。
**关键点**：豆瓣酱必须小火炒透，无生豆腥味。

3. **煮肉片**
   汤底煮开后，加入白胡椒粉、白糖、老抽、蚝油、鸡精调味。转小火，将腌好的肉片逐片或分散下入锅中，下完后静置15秒，再用勺子背轻轻推散。转回大火煮约50秒，肉片变色立刻关火，用漏勺捞出铺在配菜上。
**关键点**：小火下肉片并静置防止脱浆；煮制时间要短，保持滑嫩。

4. **浇汁**
   观察汤汁浓稠度，若不理想可用约5克淀粉加两勺水调成水淀粉，少量多次淋入锅中，边淋边推至微稠的“自来芡”状态后关火。将汤汁均匀浇在肉片上。
**注意**：芡汁不可过稠，能挂住肉片即可。

5. **泼油激香**
   在肉片顶部堆上蒜末。另起小锅烧热油至冒烟，迅速泼在蒜末和辣椒面上激香。可点缀葱花或香菜。

## 健康建议
水煮肉片富含优质蛋白与膳食纤维，但属于高油高盐菜肴，建议搭配清淡蔬菜或杂粮饭以平衡钠摄入。适合体力消耗较大的人群，肠胃敏感者需谨慎食用。
