# 泰国手标红茶
所属菜系：其他
烹饪难度：⭐⭐⭐
时令推荐：四季餐食

🧂 **食材**  
- 茶粉  
- 炼乳  
- 白糖  
- 牛奶  

🛠️ **厨具**  
- 克称  
- 带刻度容器  
- 港式奶茶过滤袋  

📋 **制作步骤**  
1. 将600毫升清水大火烧至沸腾。  
2. 取20克茶粉装入过滤袋中，将沸水冲入过滤袋中，反复过滤20次。  
3. 用克称分别称取炼乳24克、白糖24克、牛奶18毫升，置于容量不小于1000毫升的水壶中。  
4. 将过滤后的茶水倒入水壶中，搅拌至白糖完全融化。  
5. 将水壶放入冰箱冷藏4小时以上，使茶汤充分冷却并融合。  
6. 饮用前可加入6–8块冰块。

⚠️ **注意事项**  
- 水、茶粉、白糖、牛奶、炼乳的用量需严格按份量比例称量  
- 茶水需过滤足够次数（20遍）以保证茶味浓郁  
- 冷藏时间不少于4小时，确保口感绵密  
- 冰块为可选添加，饮用前加入即可

## 健康建议
炼乳、牛奶与白糖的组合使得这款泰国手标红茶含糖量较高，建议作为甜品偶尔享用。为控制糖分摄入，可减少白糖用量或使用低脂炼乳。
