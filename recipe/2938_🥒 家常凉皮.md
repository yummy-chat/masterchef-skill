# 家常凉皮
所属菜系：常见家常菜  
烹饪难度：⭐⭐⭐  
时令推荐：☀️ 夏季(4-8月)  

## 🥬 食材  
- 凉皮  
- 面筋  
- 🥒 黄瓜  
- 🧄 大蒜  
- 🌱 绿豆芽  

## 🧂 调味品  
- 🧂 盐  
- 🧅 鸡精  
- 🦪 蚝油  
- 🟤 生抽  
- 🟫 老抽  
- 🍾 香油  
- 🍶 香醋  
- 🥜 芝麻酱  

## 🍳 厨具  
- 🥣 盆  
- 🍚 碗  
- 🍽️ 盘子  
- 🧄 蒜臼  

## 📝 制作步骤  

### 准备工作  
1. 锅中注入500ml清水，大火烧沸。  
2. 下入绿豆芽，大火煮60秒，捞出过冷水，沥干后放入盘中备用。  
3. 黄瓜切细丝，码入盘中备用。  
4. 取10g蒜瓣剥皮，放入蒜臼中，加1g盐，捣成蒜泥，再加入10g清水调匀，静置备用。  
5. 市售凉皮与面筋表面可能附有食用油，用清水冲洗干净。  
6. 洗净的面筋用手轻挤，去除多余水分（无需用力挤压）。  

### 盐水调配  
1. 取小碗，加入3g盐、2g鸡精、5g生抽、1g老抽、1g香油、2g蚝油、5g香醋。  
2. 冲入25-35g温水，用筷子搅拌至盐和鸡精完全溶解，静置冷却。  

### 芝麻酱调配  
1. 将芝麻酱放入小碗中。  
2. 加入4g盐、3g鸡精、5g生抽、1g老抽、3g蚝油。  
3. 用筷子将调料与芝麻酱充分搅匀。  
4. 分2-3次加入共约10g清水，每次加后搅拌至顺滑，调至理想浓稠度。  

### 最终步骤  
1. 将凉皮放入盆中。  
2. 倒入调好的盐水，用筷子拌匀，盛入小碗，同时将盆底盐水一并加入。  
3. 在凉皮上依次放上绿豆芽和面筋。  
4. 将调好的芝麻酱从面筋上方均匀淋下。  
5. 撒上黄瓜丝。  
6. 可根据个人喜好淋入辣椒油，即可食用。  

## 健康建议  
这道家常凉皮低脂清爽，适合夏季消暑，但芝麻酱和调料中的盐分较高，需控制份量。面筋和凉皮提供少量碳水化合物，黄瓜和豆芽能补充维生素，减重期间可作一餐食用。
