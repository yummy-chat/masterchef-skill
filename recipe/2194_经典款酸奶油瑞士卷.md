# 经典款酸奶油瑞士卷
所属菜系：其他  
烹饪难度：⭐⭐⭐  
时令推荐：四季餐食  

## 食材与调味品  
🥚 蛋白 170g（冷冻至边缘结薄冰）  
🥚 蛋黄 85g  
🥛 牛奶 20g  
🌽 玉米油 50g  
🍶 酸奶油 70g（可用淡奶油等量替代）  
🌾 低筋粉 70g  
🧂 海盐 1.5g（可用普通盐替代）  
🍬 上白糖 75g（分三次加入蛋白）  
🍋 柠檬汁 约7-8滴（可用白醋替代）  
🥛 淡奶油 300g（用于夹馅）  
🍬 砂糖 18g（用于打发淡奶油）  

## 厨具与设备  
🥣 不锈钢盆（盛放蛋黄、蛋白、混合面糊）  
🥄 手动打蛋器  
⚖️ 电子秤  
🥣 小碗（混合液体）  
🕸️ 面粉筛  
🍴 硅胶刮刀  
🌡️ 食品温度计  
🔌 电动打蛋器  
🗜️ 28×28cm方形烤盘  
📄 油纸  
🪢 擀面杖  
📏 小钢尺或刮板  
🧣 冷冻抹布（包裹裱花袋）  
🧴 保鲜膜  
♨️ 烤箱  
💧 冰水盆（用于隔冰打发奶油）  
🧽 晾网  

## 制作步骤
### 蛋黄糊准备
1. 将蛋黄与蛋白分开盛放，蛋白盆放入冷冻室内，冷冻至盆壁边缘出现一圈薄冰。
   - 关键：蛋白低温打发可使组织更稳定细腻。
2. 在小碗中混合牛奶、玉米油和酸奶油，用手动打蛋器搅拌至完全融合、质地细腻。
3. 将混合液体隔水加热至约40℃。
4. 将温热的液体缓慢倒入蛋黄中，边倒边快速搅拌均匀。
5. 筛入低筋面粉，搅拌至顺滑无干粉，用刮刀刮净盆壁，然后将蛋黄糊过筛一次。
6. 蛋黄糊盆覆盖保鲜膜，置于约40℃温水中保温，使其保持在约36℃。

### 蛋白霜打发
1. 取出冰蛋白，加入海盐与柠檬汁。
2. 电动打蛋器高速打发至出现粗鱼眼泡，加入1/3的上白糖。
3. 继续高速打发至泡沫细腻，加入1/3的上白糖。
4. 继续高速打发至出现明显纹路，加入剩余的上白糖。
5. 转中低速整理气泡，打发至提起打蛋头，蛋白霜呈短小直立或顶端微弯的尖角。
   - 判断：尖角短直或微弯，不可过度打发，否则卷制时易开裂。
   - 关键：避免全程高速，低速整理可使蛋白霜组织更稳定。

### 混合蛋糕糊与烘烤
1. 取1/3蛋白霜加入保温的蛋黄糊中，用手动打蛋器快速画圈搅拌均匀。
2. 将面糊倒回剩余蛋白霜盆中，用刮刀快速翻拌几下，然后采用切拌和抄底翻拌的手法，轻快拌匀约40次，至面糊均匀有光泽、无蛋白块。
   - 关键：混合动作要快，以减少消泡，刮刀务必触及盆底。
3. 将面糊从高处倒入铺好油纸的烤盘，轻轻晃动或刮平表面，轻震两下以排出大气泡。
4. 放入预热好的烤箱中层，先以190℃烘烤10分钟，再转150℃烘烤15分钟。
   - 原理：先高温定型，再低温烤透，可避免蛋糕过度回缩。
5. 出炉后立即震盘以排出热气，趁热提着油纸将蛋糕片移至晾网上，撕开四周的油纸但保留底部，冷却至手温。

### 夹馅奶油打发与组装
1. 淡奶油加入砂糖，盆坐于冰水上，电动打蛋器先高速打发至出现纹路，再低速打发至硬挺，提起打蛋头出现直立尖角。
   - 必须隔冰水打发，否则温度过高易导致失败。
2. 将冷却好的蛋糕片翻面，放在新油纸上，两端斜切45°角。
3. 将打发好的奶油装入裱花袋，袋外包裹冷冻抹布以防手温使奶油融化。
4. 在靠近身体一侧约1cm处挤数道奶油，堆成“小山丘”作为中心馅料，其余部分抹平奶油。
5. 用擀面杖辅助油纸，从有山丘的一端一气呵成卷起。
6. 用小钢尺或刮板抵住底部，同时向内拉紧油纸以收紧蛋糕卷，再用油纸包好。

### 冷藏定型
将包好的蛋糕卷放入冰箱冷藏至少1小时，取出后撕掉油纸，切去两端不规整部分，即可切块享用。

## 健康建议  
这道酸奶油瑞士卷采用低温蛋白霜与变温烘烤技巧，保证了蛋糕体的柔韧与细腻，适合作为下午茶甜点。注意蛋白切勿打发过度，否则卷制时易开裂。此外，夹馅奶油务必隔冰水打发，以免夏季高温导致失败。
