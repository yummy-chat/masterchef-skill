# 五指毛桃茯苓薏米茶
所属菜系：粤菜  
烹饪难度：⭐  
时令推荐：春季(1-4月)  

## 食材与调味品  
🥕 五指毛桃 20g / 40g / 50g（分别对应750ml / 1500ml / 2000ml水）  
🍄 茯苓 10g / 20g / 25g（分别对应750ml / 1500ml / 2000ml水）  
🌾 熟薏米 10g / 20g / 25g（分别对应750ml / 1500ml / 2000ml水）  
🍊 陈皮 3g / 5g（分别对应750ml / 1500ml / 2000ml水）  
🍬 冰糖 适量（可选，煮好后添加）  
🍯 蜂蜜 适量（可选，煮好后添加）  

## 厨具与设备  
🫖 玻璃养生壶 / 煮水壶  
🔥 电磁炉  
🥣 碗（用于清洗食材）  

## 制作步骤
1. 根据饮用人数选择对应水量，按比例称取五指毛桃、茯苓、熟薏米、陈皮。
**注意**：若只有生薏米，需用干锅小火煸炒至表面微黄、散发香气，制成熟薏米后使用。

2. 将所有干食材用清水冲洗干净，陈皮掰碎后再冲洗。
3. 把洗好的食材放入玻璃壶，加入对应比例的水。大火煮开后转小火，煮30分钟，至汤色变为金黄、散发谷物和椰奶香气即可。
**关键点**：此为简单版配方，口感自然清甜。

4. 关火后可视个人喜好加入冰糖或蜂蜜调味，搅拌均匀后饮用。

## 健康建议  
这款祛湿健脾茶适合春季湿气重、消化不良的人群，但孕妇及体质虚寒者慎用。建议不加糖饮用，以保留五指毛桃天然的椰奶清香。
